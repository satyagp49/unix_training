          Good article. One important correction concerning the socket type description:

          In your socket() system call, you must specify AF_UNIX, not AF_INET.  Although structurally similar, Unix domain sockets are just that: for use within the local Unix system domain only as visible entity on the file system, they do not extend to any network transport layer.
